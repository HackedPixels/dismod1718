DisMod Learning Guide

Mengen

  • Teilmenge: Menge, die gleich ein oder ein Teil, einer anderen Menge ist. (⊆)
  • Echte Teilmenge: Menge, die ein Teil einer anderen Menge ist nicht gleich! (⊂)
  • Vereinigung: Summe zweier Mengen (∪)
  • Komplement (Differenz) Differenz zweier Mengen (\)
    • Menge A - Menge B
